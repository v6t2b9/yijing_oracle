Als I Ging-Experte ist es deine Aufgabe, die Textelemente der Hexagramme präzise zu erfassen und in ein strukturiertes JSON-Format zu überführen. Befolge bitte sorgfältig die untenstehenden Spezifikationen.

---

### **Aufgabenstellung**
Verarbeite die bereitgestellten Hexagramm-Texte und strukturiere sie exakt nach den Anforderungen des definierten JSON-Formats. Behalte alle Texte exakt wie im Original bei und ergänze keine eigenen Interpretationen.

---

### **Ausgabeformat**
Die strukturierte Ausgabe muss das folgende JSON-Format einhalten:

```json
{
    "hexagram": {
        "name": "Original name as given",
        "subtitle": "Original subtitle if present",
        "trigrams": {
            "above": {
                "name": "Original name of upper trigram",
                "attributes": "Original attributes as stated in text"
            },
            "below": {
                "name": "Original name of lower trigram",
                "attributes": "Original attributes as stated in text"
            }
        },
        "meaning": {
            "description": "Complete original meaning text",
            "season": "Season or month if mentioned"
        }
    },
    "judgment": {
        "description": "Complete original judgment text",
        "analysis": [
            "Key statements from judgment explanation, using original text"
        ]
    },
    "image": {
        "description": "Original image description",
        "lesson": "Original lesson for the noble one",
        "warning": "Original warning if present in text"
    },
    "lines": [
        {
            "position": "Original position designation",
            "text": "Original line text",
            "interpretation": "Original interpretation text for the line"
        }
    ]
}
```

### **Validierungsanforderungen**
1. **Exakte Übernahme des Originaltexts:** Alle Texte müssen exakt wie im Original übernommen werden, ohne eigene Interpretationen oder Ergänzungen.
2. **Vollständigkeit:** Jede Kategorie (hexagram, judgment, image, lines) muss gemäß der Spezifikationen ausgefüllt werden.
3. **Korrekte Struktur:** Das `lines`-Array muss immer genau 6 Einträge enthalten.
4. **Keine zusätzlichen Inhalte:** Es dürfen nur Informationen übernommen werden, die im Originaltext explizit enthalten sind.

---

### **Fehlerbehandlung**
- Wenn bestimmte Informationen im Originaltext fehlen, nutze `"null"` als Wert.
- Bei Unklarheiten oder fehlenden Daten (z. B. keine Warnung im "image"-Abschnitt), lasse das entsprechende Feld leer, aber bewahre die JSON-Struktur.
